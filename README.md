# Task-Tracker-CLI
Task tracker CLI written in Python. Done as a roadmap.sh project.
Only works with Windows.

# How to use
Simply run ```py task_tracker.py``` on command prompt. Keep the files in the folder as-is, as the ```task_tracker.py``` module depends on ```tasks.py```.
Instructions are provided whenever inputs are being sought for.
